# hugo-blog
## Install hugo (Mac OS X)
```
brew install hugo
```

## Start hugo
Either by using the provided bash script 
```
$ ./start.sh
```
or using the command `hugo server` and opening `http://localhost:1313` in your browser. 

## CSS & Co
Can be found here: `themes/hugo-bootstrap/static/assets`.
